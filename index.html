Modelo de Cores Aditivo e Subtrativo

Modelo Aditivo (RGB)

O modelo aditivo de cores é baseado na combinação de luzes coloridas. As cores primárias neste modelo são o vermelho (Red), verde (Green) e azul (Blue), conhecidas como RGB. Quando essas cores são combinadas em diferentes intensidades, elas produzem outras cores. Por exemplo:

  • Vermelho + Verde = Amarelo
  • Vermelho + Azul = Magenta
  • Verde + Azul = Ciano
  • Vermelho + Verde + Azul = Branco

A intensidade da luz influencia diretamente a cor final. Quanto maior a intensidade, mais clara será a cor. Em telas de dispositivos eletrônicos, cada pixel é composto por subpixels RGB, que ajustam sua intensidade para criar a cor desejada.

Modelo Subtrativo (CMYK)

O modelo subtrativo de cores é utilizado principalmente em impressão e é baseado na mistura de pigmentos. As cores primárias neste modelo são o ciano (Cyan), magenta (Magenta), amarelo (Yellow) e preto (Key), conhecidas como CMYK. O preto é adicionado para gerar contraste e reduzir o desperdício de tinta.

Exemplos de combinações:

  • Ciano + Magenta = Azul
  • Magenta + Amarelo = Vermelho
  • Ciano + Amarelo = Verde
  • Ciano + Magenta + Amarelo = Preto (aproximado)

A cor preta (K) é essencial para melhorar a qualidade das impressões, pois permite um preto mais puro e reduz a quantidade de tinta necessária.

Comparação entre os Modelos

O modelo aditivo (RGB) é utilizado em telas de dispositivos eletrônicos, onde a luz é emitida diretamente. Já o modelo subtrativo (CMYK) é usado em impressão, onde a luz é refletida a partir de superfícies. Enquanto o RGB combina luzes para criar cores, o CMYK subtrai luz ao misturar pigmentos.

Por exemplo, em uma tela de computador, a combinação de RGB em máxima intensidade produz branco, enquanto na impressão, a combinação de CMY produz um preto aproximado, que é aprimorado com a adição de preto (K).
